## Java

las2peer uses **Java 14**.


## Build

Execute the following command on your shell:

```bash
gradle clean build
```

## Start

To start this service, use one of the available start scripts:

Windows:

```
bin/start_network.bat
```

Unix/Mac:

```
bin/start_network.sh
```
